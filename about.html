About This Training Site

Purpose

This demo site is designed to teach Google Analytics 4 and Google Tag Manager implementation in a real-world environment. Unlike platform-specific builders, this gives you complete control over the tracking implementation.

What Makes This Different?

  • Full access to HTML head and body tags
  • Clean, semantic markup for reliable CSS selectors
  • Real forms, buttons, and interactive elements
  • Ability to intentionally break tracking for debugging practice
  • Version control through GitHub for audit trails

Training Scenarios

Scenario 1: Basic Setup

Install GTM container, verify pageviews, configure basic events

Scenario 2: Form Tracking

Track newsletter signups and contact form submissions

Scenario 3: E-commerce Events

Implement view_item, add_to_cart, and purchase events

Scenario 4: File Downloads

Track PDF downloads and resource engagement
